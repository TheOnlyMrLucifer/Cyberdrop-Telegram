## Getting Started

```sh
# clone it
git clone git@github.com:TheOnlyMrLucifer/Cyberdrop-Telegram.git

# Make it your own
rm -rf .git && git init && npm init

# Install dependencies
npm install

# Start development live-reload server
npm run dev

# Start production server:
npm start
```
